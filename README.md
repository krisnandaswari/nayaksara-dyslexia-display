# Nayaksara — Dyslexia-Tested Display Sans Serif Typeface

Nayaksara is a display sans-serif typeface designed with consideration for readers with dyslexia, focusing on improved letter distinction, balanced proportions, and comfortable spacing.

This typeface was independently created by the author as part of an internship program and is released as an open-source project under the SIL Open Font License 1.1.

---

## Important Accessibility Notice

Nayaksara **is not an absolute or universal dyslexia font**.

The typeface has been **tested with users who have dyslexia and has demonstrated supportive readability**, but reading experiences can vary between individuals.

Nayaksara should be understood as:
- A dyslexia-considerate typeface  
- A visual communication tool with accessibility awareness  
- **Not a medical, therapeutic, or definitive dyslexia solution**
