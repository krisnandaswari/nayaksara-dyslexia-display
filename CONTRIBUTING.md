# Contributing Policy

Thank you for your interest in the Nayaksara typeface project.

This project is open-source under the SIL Open Font License 1.1 and follows a **curated, non-contributive development model**.

---

## External Contributions

Please note:

- ❌ Pull requests are not accepted
- ❌ External design or code contributions will not be merged
- ❌ Feature implementation requests are not accepted

This policy ensures:
- Clear authorship and responsibility
- Consistent design vision
- Respect for the project’s educational origin

---

## Permitted Use

You may:
- Study the source files
- Fork the repository for personal exploration
- Modify the font for your own use under the OFL
- Redistribute modified versions under a different name, as required by the OFL

---

## Feedback

- Feedback may be shared through GitHub Issues
- Issues are for discussion only and do not imply future changes

---

## Attribution

All authorship and final design authority remain with the original designer.

Thank you for respecting this contribution model.
